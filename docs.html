Editing the Docs

Editing the docs

There are a few prerequisites to being able to edit this documentation.

In order to push any documentation changes, you'll need gitbook.

$ npm install -g gitbook-cli

The documents are in blacksheepdesign/webpack-bsd:docs. To make changes, simply edit the Markdown files, and run the following script to deploy the docs.

$ ./deploy-docs.sh
